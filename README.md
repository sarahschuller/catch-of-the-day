# catch-of-the-day
 
<h1>Features</h1>
<ul>
 <li>
 Users can log in via GitHub or Google to gain access to store inventory.
</li>
 <li>
  Store owners can add and remove fish from the inventory.
 <li>
  Non-owner users can view the store and add fish to their order.
 </li>
</ul>
 
<h1>Tools</h1>
 <ul>
  <li>
   React
  </li>
  <li>
   Firebase
  </li>
 </ul>
